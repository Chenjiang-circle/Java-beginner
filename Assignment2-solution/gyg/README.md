> ## Java支持的数据类型有哪些？什么是自动拆装箱？
> + 1.基本数据类型
> +Java基本类型共有八种，基本类型可以分为三类：
> 字符类型char
> 布尔类型boolean
> 整数类型byte、short、int、long
> 浮点数类型float、double。
> #### Java中的数值类型不存在无符号的，它们的取值范围是固定的，不会随着机器硬件环境或者操作系统的改变而改变。
